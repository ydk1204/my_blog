## 리액트와 CSS

우리가 리액트에서 css를 사용할 때 방법은 여러가지가 존재한다.

1. global.css에 다 우겨넣기
2. .module.css로 모듈화 시켜서 원하는 파일에만 적용 시키기
3. css 라이브러리 사용하기 등

사람들은 정말 다양한 방법으로 스타일링을 하는데 나는 주로 module이나
tailwindCSS를 사용하는 편이다.  
그런데 이런 css를 사용할 때 변수를 사용하면 재사용면에서 편하고
보기 쉽기 때문에 가끔씩 사용하는 편인데, 이런 css변수를
리액트에서는 어떤 방식으로 쓰이는지 알아보자.

---

## CSS 변수

리액트에서 css변수를 할당하는 방법은 매우 간단하다.

방식은 다음과 같다.
먼저 js파일에서 다음과 같은 코드를 작성한다.

```javascript
const color = {
  "--clr": "#FF8000",
};

<div style={color}>content</div>;
```

1. color는 css 변수를 담은 객체형태의 변수이다.
2. 객체에 key와 value를 지정하는데 key는 css에서 사용할 변수 이름, value는 사용자 지정 값이다.
3. 내가 원하는 요소 내에 css 변수를 style 속성의 값으로 할당한다.

그 다음 css 파일에서는

```css
/* ...다른 스타일링 코드  */
background: var(--clr);
```

1. 원하는 속성에 해당 요소에 할당된 css 변수 이름을 지정한다.

css 파일에서는 일반적인 css변수 사용과 동일한 방법으로 사용하면 된다.
