네트워크 통신에는 몇 가지 방식이 있다. 각각의 개념을 알아보자

# 1.유니캐스트(Unicast)

- 유니캐스트는 **1:1 통신**이다.
즉, **출발지와 목적지가 1:1 관계를 가진다**.

- 유니캐스트는 단일 대상에게 데이터를 보내는 방식이다.

- IPv4 주소에서는 A, B, C 클래만 유니캐스트 형태를 지원한다.
멀티캐스트 형태는 클래스 D 주소를 사용한다.

- 예를 들어 **개별 컴퓨터 간의 통신**을 할 때 사용된다.

---

# 2.멀티캐스트(Multicast)

- **1:다수 그룹 통신**이다. 특정 그룹에 속한 시스템에게만 정보를 전송한다.

- IPv4 주소에서는 클래스 D 주소 범위(224.0.0.0 ~ 239.255.255.255)를 사용한다.

- 멀티캐스트는 IP 멀티캐스트 라우터를 통해 여러 네트워크 집합을 넘나드는 경우에 사용된다.

- 예를 들어 **영상회의, IP TV, 인터렉티브 게임**

---

# 3.브로드캐스트(Broadcast)

- **로컬 LAN**에 연결된 **모든 시스템에게 프레임**을 보내는 방식

- 브로드캐스트 주소로는 255.255.255.255 (리미티드 브로드캐스트 주소) 또는 IP 주소의
호스트부를 1로 만든 주소(디렉티드 브로드캐스트 주소)를 사용한다.

- 주로 **ARP나 DHCP**와 같은 프로토콜에서 사용된다.
